# Анализ датасета HeadHunter_Vacancies :office:

## Задачи
- Провести первичный анализ датасета
- Проанализировать вакансии без опыта (посмотреть какие навыки более востребованы, какая медианая з/п нижняя и верхняя, оценить стоит ли входить в IT, посмотреть каких специалистов действительно нехватает)
- Построить графики при помощи библиотеки matplotlib
- Ответить на вопрос: "Какие навыки прокачивать начинающему специалисту?"
- Ответить на вопрос: "На какую зарплату расчитывать начинающему специалисту?"
